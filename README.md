## 動作確認方法

### 環境構築
1. リポジトリをクローンする
2. 以下のライブラリをインストールする
   1. Django
3. `python3 manage.py migrate`コマンドを入力し、マイグレーションを行う
4. `python3 manage.py runserver`コマンドでサーバーを立ち上げる
